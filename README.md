# Todo CLI

## Features
  - Run `todo add <...items>` to add items
  - Run `todo list` to list items for the current day
  - Run `todo mark <...tasks>` to mark of tasks to not appear when `todo list` is run

## Quick Start

### Prequisites
  - A C++ Compiler supporting C++23
  - Make
  - A unix system (unless your willing to write your own build file for windows)
  - A bash interpreter

### Installation

To install run:
```bash
git clone https://github.com/flappibird123/todo-cli.git
cd todo-cli
chmod +x build.sh
./build.sh
```

## Usage
  Run `todo` to start
